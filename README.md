# DungeonCard_Core

> 毕设项目《地牢卡牌》核心战斗系统源代码（Unity / C#）

---

## 📖 项目简介

独立开发的一款类《杀戮尖塔》风格的卡牌 Roguelike 游戏核心战斗模块。

本仓库包含以下核心功能：
- 配置驱动的多关卡流程系统（固定 + 随机分支）
- 动态敌人加载与智能防碰撞生成
- 有限状态机（FSM）管理的回合制战斗流程
- 协程驱动的敌人 AI 串行执行与关卡平滑过渡
- 单例模式与模块解耦架构

---

## 🛠 技术栈

- Unity 3D
- C# 语言
- 设计模式：单例（Singleton）、状态机（FSM）、配置驱动（Data-Driven）

## 📁 核心代码文件

- `EnemyManager.cs`：全局战斗管理器，负责关卡加载、敌人生成、战斗流程控制
- `FightCardManager.cs`：
- 基于**双堆结构**（手牌堆 / 弃牌堆）实现完整抽卡循环
- 手牌堆为空时**自动回流弃牌堆**并洗牌，保证战斗永不卡死
- 跨关卡**卡组继承**机制，保留玩家成长进度（Roguelike 风格）
- 使用 **Fisher-Yates 洗牌算法**保证真随机分布
